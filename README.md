# 🌐 Region Website

A responsive web application designed to showcase tourism in the Upper Northeastern region of Thailand. The website covers 11 provinces and provides information about tourist attractions, local restaurants, and regional products, with a focus on clean UI and user-friendly experience.

---

## ✨ Features
- 📍 Display tourism information across 11 provinces in Upper Northeastern Thailand  
- 🏞️ Highlight tourist attractions, local restaurants, and local products  
- 🎨 Clean and simple UI design  
- 📱 Responsive layout (mobile-friendly)  

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

---

## 🎯 Project Purpose
This project was developed to practice frontend development and improve UX/UI design skills by creating a simple and accessible tourism website that promotes regional content.

---

## 🚀 How to Run
1. Download or clone this repository  
2. Open `Main.html` in your browser
